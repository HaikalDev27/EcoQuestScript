
# EcoQuest Editable Scrip

A 2D Game made in unity where you need to discover something new to win


## Tech Stack

**Language:** C#

**Server:** Node, Express

## Features

- Basic movement mechanism
- Interactable object
- Simple inventory system
- Ending mechanism
